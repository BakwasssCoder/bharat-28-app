# BHARAT 28 Admin Panel Usage Guide

## Getting Started

1. Navigate to `/admin` in your browser
2. Enter your admin credentials
3. Click "Access Admin"

## Dashboard Overview

The admin dashboard has two main sections:

### 1. Billing Section
- Create new orders at the counter
- Add menu items to orders
- Select payment mode (Cash, UPI, Card)
- Process orders and generate bills

### 2. Sales Dashboard
- View today's sales statistics
- See weekly and monthly sales data
- Review recent orders
- Access sales reports

## Billing Process

### Creating a New Order

1. Go to the "Billing" tab
2. Browse menu items on the left panel
3. Click the "+" button next to items to add them to the cart
4. Adjust quantities using the +/- buttons in the cart
5. Select payment mode from the dropdown
6. Click "Charge â‚¹[amount]" to process the order

### Managing Cart Items

- **Increase Quantity**: Click the "+" button
- **Decrease Quantity**: Click the "-" button
- **Remove Item**: Click the trash icon
- **Clear Cart**: Refresh the page or logout and login again

## Sales Dashboard

### Statistics Cards

- **Today's Orders**: Number of orders processed today
- **Today's Revenue**: Total revenue generated today
- **Weekly Orders**: Number of orders in the past week

### Recent Orders

View the 5 most recent orders with:
- Order number
- Date
- Total amount
- Payment mode

## Security Features

- Session timeout after inactivity
- JWT-based authentication
- Encrypted password storage
- Role-based access control

## Best Practices

### Order Management

1. Always double-check items and quantities before processing
2. Confirm payment mode with the customer
3. Keep a physical backup of important orders
4. Regularly review sales reports for business insights

### Account Security

1. Use strong, unique passwords
2. Change passwords regularly
3. Log out when leaving the computer unattended
4. Report suspicious activity immediately

## Troubleshooting

### Login Issues

- **Forgot Password**: Contact system administrator to reset
- **Account Locked**: Contact system administrator
- **Invalid Credentials**: Double-check username and password

### Billing Issues

- **Cannot Add Items**: Refresh the page and try again
- **Processing Errors**: Check internet connection and try again
- **Duplicate Orders**: Verify order was not already processed

### Reporting Issues

- **Missing Data**: Check date ranges and filters
- **Incorrect Totals**: Verify data entry and calculation methods
- **Slow Performance**: Clear browser cache and refresh
